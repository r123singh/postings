Follow the below the steps to use the source code and contribute in the blog:

1. Download the "template.html" file to your local machine
2. Open the template file in notepad++ or editing tool.
3. Replace the place holder text in CAPITAL LETTERS with your training blogs/articles.
4. Save the template file with name "blog.html"
5. Add file to a folder along with supporting image file names. 
6. Upload compressed folder to this Github repository
 

Thanks for downloading this template and contributing to the source!!
